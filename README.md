# Advanced LOD Generator by Troneks

An advanced Blender addon for generating Levels of Detail (LOD) with custom settings and multi-language support.

## Installation Instructions

1. **Download the Addon:**
   - Download the `advanced_lod_generator.py` file from this repository.

2. **Install the Addon in Blender:**
   - Open Blender.
   - Go to `Edit` -> `Preferences...` -> `Add-ons`.
   - Click on the `Install...` button at the top.
   - Navigate to where you saved `advanced_lod_generator.py` and select it.
   - Click `Install Add-on`.

3. **Enable the Addon:**
   - After installation, search for "Advanced LOD Generator" in the add-ons list.
   - Check the box next to the addon name to enable it.

4. **Use the Addon:**
   - Open the 3D Viewport.
   - Press `N` to open the sidebar.
   - Navigate to the `LOD Generator` tab.
   - Configure your settings and generate LODs as needed.

## Advantages of the Addon

- **Customizable LOD Generation:**
  - Generate multiple LODs with precise control over the reduction percentage for each level.
  - Supports up to 5 LOD levels.

- **User-Friendly Interface:**
  - Intuitive controls integrated directly into Blender's sidebar.
  - Multi-language support: English, Russian, Chinese, and Spanish.

- **Advanced Mesh Simplification:**
  - Efficient algorithms reduce polygon count while preserving essential details.
  - Maintains UV maps and materials.

- **Flexible Naming Conventions:**
  - Customize LOD object names with user-defined templates.
  - Use placeholders like `{original_name}` and `{index}`.

- **Automatic LOD Offset:**
  - Option to shift generated LODs along a chosen axis to prevent overlap.
  - Simplifies comparison and visualization of different LOD levels.

- **No Modifier Stack Clutter:**
  - Generates simplified meshes without adding modifiers, keeping your modifier stack clean.
